# Start Here - BuilderHub Mobile

## Quick Start

This is a **COMPLETELY NEW** Flutter app with BuilderHub branding and Burkina Faso colors.

### 1. Install Dependencies

```bash
cd builderhub_mobile
flutter pub get
```

### 2. Run the App

```bash
flutter run
```

## What You'll See

### Auth Screen
- **Red background** with BuilderHub logo
- Login/Signup toggle
- User type selection (Client or Artisan)
- Green and Yellow accents

### Home Screen
- **Red AppBar** with "BuilderHub" title
- Category cards in **Green, Yellow, and Red**
- Top-rated artisans section
- Bottom navigation with 4 tabs
